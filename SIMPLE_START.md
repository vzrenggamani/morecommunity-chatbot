# SUPER SIMPLE SETUP

## 🚀 Quick Start (3 steps):

### 1. Add your API key
Create a `.env` file:
```
GOOGLE_API_KEY=your_actual_api_key_here
```

### 2. Run locally
```bash
# Windows
start.bat

# Linux/Mac
chmod +x start.sh && ./start.sh
```

### 3. Open browser
Go to: http://localhost:8501

## 🐳 Docker (if you want)
```bash
docker-compose -f docker-compose.simple.yml up
```

That's it! 🎉
